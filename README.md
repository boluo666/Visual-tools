## 使用方法

1. 粘贴你想下载的图片 URL  
2. 点击 **生成链接 / Generate Links / Buat Tautan**  
3. 等待各代理测速完成  
4. 点击 **Download** 下载图片，或点击 **Open** 在新标签打开  
5. 查看延迟选择最快的代理下载

---
